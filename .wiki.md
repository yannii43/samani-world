# Project Summary
Sanii World is a premium e-commerce platform tailored for the Senegalese market, specializing in an exclusive collection of women's fashion, including elegant dresses, abayas, and accessories. The platform is designed with a luxurious and minimalistic aesthetic, prioritizing user experience through intuitive navigation and seamless transactions. It includes a secure authentication system for customers and administrators, facilitating effective product, order, and payment management.

# Project Module Description
- **Homepage**: Displays a hero section, new arrivals, best-sellers, and product categories.
- **Product Catalog**: Users can browse products with dynamic filters and sorting options.
- **Product Detail Page**: Provides comprehensive product information, including images, descriptions, pricing, and variant selections.
- **Shopping Cart**: Allows users to manage selected items, adjust quantities, and view total costs.
- **Checkout Process**: Collects user information and offers various payment and delivery options.
- **Authentication**: 
  - **Login Page**: Secure access for customers and administrators.
  - **Registration Page**: Allows customers to create accounts with validation.
  - **Profile Page**: Enables users to update personal information and view order history.
- **Admin Dashboard**: 
  - **Dashboard**: Overview with statistics on products, orders, and revenue.
  - **Product Management**: Admins can add, modify, and delete products and variants.
  - **Category Management**: Admins manage product categories and subcategories.
  - **Order Management**: Admins can view and update order statuses.
- **About Page**: Information about the store and delivery zones.
- **Order Confirmation Page**: Displays order details and confirmation message.

# File Description Inventory
- **README.md**: Overview of the project and setup instructions.
- **MIGRATION_PLAN.md**: Detailed plan for transforming DakarShop into Sanii World.
- **package.json**: Lists dependencies and scripts for building and running the project.
- **src/**: Contains the source code, including components, pages, and styles.
- **public/**: Contains static assets such as images and favicon.
- **components/**: Contains UI components like buttons, cards, headers, and dropdowns.
- **pages/**: Contains main application pages including product catalog, product detail, cart, checkout, login, register, profile, and admin dashboard.
- **lib/**: Contains utility functions and data management for products, orders, shipping, and coupons.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Typed superset of JavaScript for improved development experience.
- **Vite**: Build tool for fast development and optimized production builds.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Zustand**: State management library for managing authentication and shopping cart state.
- **React Router**: For routing and navigation within the application.
- **TanStack Query**: For data fetching and state management.

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run start
   ```
