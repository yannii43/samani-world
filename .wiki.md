# Project Summary
DakarShop is an e-commerce platform tailored for the Senegalese market, offering a wide variety of products, including electronics, fashion, and home goods. The platform is designed to enhance user experience with a modern, responsive layout, facilitating seamless online shopping and efficient delivery services within Dakar. Recently, a comprehensive authentication system has been implemented to support both customer and admin accounts, ensuring secure access and management of products and payments.

# Project Module Description
- **Homepage**: Showcases a vibrant hero section, popular categories, and featured products.
- **Product Catalog**: Users can filter and sort products by category and price.
- **Product Detail Page**: Provides detailed product information, including images, descriptions, and pricing.
- **Shopping Cart**: Enables users to manage their selected items, adjust quantities, and view total costs.
- **Checkout Process**: Collects user information for delivery and payment options.
- **Authentication**: 
  - **Login Page**: Secure login for customers and administrators.
  - **Registration Page**: Allows customers to create accounts with validation.
  - **Profile Page**: Users can update personal information and view order history.
- **Admin Dashboard**: 
  - **Dashboard**: Overview with statistics (products, orders, revenue).
  - **Product Management**: Admins can add, modify, and delete products.
  - **Order Management**: Admins can view and update order statuses.
- **About Page**: Background information about the store and delivery zones.

# File Description Inventory
- **README.md**: Overview of the project and setup instructions.
- **package.json**: Lists dependencies and scripts for building and running the project.
- **src/**: Contains the source code, including components, pages, and styles.
- **public/**: Contains static assets such as images and favicon.
- **components/**: Contains UI components like buttons, cards, headers, and dropdowns.
- **pages/**: Contains the main application pages such as the product catalog, product detail, cart, checkout, login, register, profile, and admin dashboard.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Typed superset of JavaScript for better development experience.
- **Vite**: Build tool for fast development and optimized production builds.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Zustand**: State management library for managing authentication and shopping cart state.
- **React Router**: For routing and navigation within the application.

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run start
   ```
