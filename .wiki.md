# Project Summary
Sanii World is a premium e-commerce platform focused on the Senegalese market, offering an exclusive selection of women's fashion products, including elegant dresses, abayas, and accessories. The platform emphasizes a luxurious and minimalistic design, enhancing user experience through intuitive navigation and seamless transactions. It features a robust authentication system for both customers and administrators, ensuring secure access to manage products, orders, and payments.

# Project Module Description
- **Homepage**: Features a hero section, new arrivals, best-sellers, and product categories.
- **Product Catalog**: Allows users to explore products with dynamic filters and sorting options.
- **Product Detail Page**: Displays detailed product information, including images, descriptions, pricing, and variant selection.
- **Shopping Cart**: Users can manage selected items, adjust quantities, and view total costs.
- **Checkout Process**: Collects user information and offers various payment and delivery options.
- **Authentication**: 
  - **Login Page**: Secure login for customers and administrators.
  - **Registration Page**: Enables customers to create accounts with validation.
  - **Profile Page**: Users can update personal information and view order history.
- **Admin Dashboard**: 
  - **Dashboard**: Overview with statistics on products, orders, and revenue.
  - **Product Management**: Admins can add, modify, and delete products and variants.
  - **Category Management**: Admins can manage product categories and subcategories.
  - **Order Management**: Admins can view and update order statuses.
- **About Page**: Provides background information about the store and delivery zones.

# File Description Inventory
- **README.md**: Overview of the project and setup instructions.
- **MIGRATION_PLAN.md**: Detailed plan for transforming DakarShop into Sanii World.
- **package.json**: Lists dependencies and scripts for building and running the project.
- **src/**: Contains the source code, including components, pages, and styles.
- **public/**: Contains static assets such as images and favicon.
- **components/**: Contains UI components like buttons, cards, headers, and dropdowns.
- **pages/**: Contains the main application pages such as the product catalog, product detail, cart, checkout, login, register, profile, and admin dashboard.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Typed superset of JavaScript for better development experience.
- **Vite**: Build tool for fast development and optimized production builds.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Zustand**: State management library for managing authentication and shopping cart state.
- **React Router**: For routing and navigation within the application.

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run start
   ```
