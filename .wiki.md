# Project Summary
DakarShop is an e-commerce platform designed for the Senegalese market, offering a diverse range of products including electronics, fashion, and home goods. The platform prioritizes user experience with a modern, responsive design and aims to facilitate easy online shopping and fast delivery within Dakar.

# Project Module Description
- **Homepage**: Features a vibrant hero section, popular categories, and featured products.
- **Product Catalog**: Users can filter and sort products by category and price.
- **Product Detail Page**: Displays comprehensive product information, including images, descriptions, and pricing.
- **Shopping Cart**: Allows users to manage their selected items, adjust quantities, and view total costs.
- **Checkout Process**: Collects user information for delivery and payment options.
- **About Page**: Provides background information on the store and delivery zones.

# File Description Inventory
- **README.md**: Overview of the project and setup instructions.
- **package.json**: Lists dependencies and scripts for building and running the project.
- **src/**: Contains the source code, including components, pages, and styles.
- **public/**: Contains static assets such as images and favicon.
- **components/**: Contains UI components like buttons, cards, and headers.
- **pages/**: Contains the main application pages such as the product catalog, product detail, cart, and checkout.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Typed superset of JavaScript for better development experience.
- **Vite**: Build tool for fast development and optimized production builds.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Zustand**: State management library for managing the shopping cart state.
- **React Router**: For routing and navigation within the application.

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run start
   ```
